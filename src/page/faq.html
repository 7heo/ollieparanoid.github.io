About & FAQ

I'll be writing more about postmarketOS and related topics, such as Alpine Linux and maybe Halium on this blog. But also about experiments with crowdfunding the whole thing to make it my next day job. Feel free to write me more questions.

Any plans to collaborate with Halium?

Yes, I hope we can share udev rules, unify kernels and maybe work on a few other parts together! But keep in mind, that postmarketOS has a different architecture (mostly due to Alpine Linux, which is required for pmbootstrap), so it can not build on the Halium stack and this will not be a goal in the future. It does not use the glibc, systemd or the Android build system, or run Android next to GNU/Linux.

Why did you wait so long before releasing anything?

I needed to make sure, that pmbootstrap is safe to use. It downloads Alpine's apk (Alpine Linux package manager, not to be confused with the Android app format) binary from the Internet and executes it with root rights. This means, that apk needs to be verified properly before execution (GPG key signature check with openssl, minimum required version), and the whole verification process needed to have testcases (which it does now). Furthermore I took my time to put up this blog and write the announcement, as well as the rest of the content.

Oh and I did actually release my Alpine cross-compiling efforts about a month earlier.

Why do you keep talking about GNU/Linux all the time, when Alpine uses musl and BusyBox instead of glibc and the GNU Core Utilities?

Alpine has replaced a lot of GNU components, but still uses GCC and the binutils. They have also packaged just about every typical GNU userspace component, except for the glibc so you can just install them if you want to (example: apk add bash). Besides that, I personally like the GNU philosophy and calling postmarketOS a GNU/Linux distribution helps to differentiate it better:
  • GNU/Linux (Linux kernel with more or less GNU userspace)
  • Android/Linux (Linux kernel with Android userspace)
  • Linux (just the kernel)
